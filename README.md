# Miuros
Miuros.com - Marketing Website

## Packages
Install Packages
`yarn install` 


## Gatsby
Install Gatsby
`npm i -g gatsby-cli`

## Contentful

Add `.contentful.json`

```
{
  "spaceId":"...",
  "accessToken":"..."
}
```
To run dev environment
`yarn run dev` 

To run build condition & test
```
gatsby build

gatsby serve
```
